# homebrew-orla-test

TEST TAP - DO NOT USE IN PRODUCTION

This is a test Homebrew tap for [Orla](https://github.com/dorcha-inc/orla) used for CI/CD testing purposes.

## What is this?

This repository contains the Homebrew cask for Orla, which is automatically maintained by [GoReleaser](https://goreleaser.com) during test releases (when `TEST_MODE=true`).

This tap is for testing only and should not be used by end users. The production tap is at [dorcha-inc/homebrew-orla](https://github.com/dorcha-inc/homebrew-orla).

Do not manually edit files in this repository. All changes are automated.
